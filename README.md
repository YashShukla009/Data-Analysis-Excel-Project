# Data-Analysis-Excel-Project
Analyzed 300+ consumer purchase records using Excel to identify sales trends, customer behavior, and payment preferences through pivot tables and an interactive dashboard.

Project Overview
This project focuses on analyzing consumer purchase behavior using Excel.
The dataset contains 300+ consumer transaction records across multiple cities, categories, and payment modes.
The goal is to clean raw data, perform analysis using pivot tables, and build an interactive Excel dashboard to generate business insights.

Business Questions
Which product categories generate the highest revenue?
Which cities contribute the most to overall sales?
What is the monthly sales trend over time?
Which payment modes are most preferred by customers?
Which customer segment shows the highest average purchase value?

Business Insights
Electronics and Furniture generate the highest revenue
Metro cities contribute the majority of total sales
Credit Card is the most preferred payment mode
Young Adults show the highest average spending behavior
